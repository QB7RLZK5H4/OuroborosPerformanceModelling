# Ouroboros Performance Modelling

These files are supporting material for the paper "Towards Validated Performance Modelling of
Ouroboros Blockchain Protocols" submitted to [MARBLE 2026](https://www.marble-conference.org/marble2026-cfp).

* IHaskell notebook of the ∆Q model presented in Sections 3 and 4 of the paper: [PraosTransactionModel.ipynb](PraosTransactionModel.ipynb)

* Wolfram/Mathematica code for Bulk Service Model: BulkServiceModel.nb (this file cannot be displayed in GitHub)
  
* CSV files and PNG files output from the Bulk Service Model: [BSM_results](BSM_results)
     * BulkServiceResultsCap_k**X**p_N176: files for k = **X**
     * BulkServiceResultsCap_k**X**p5_N176: files for k = **X**.5

* Haskell files for plots from the extended ∆Q models: [src](src)
